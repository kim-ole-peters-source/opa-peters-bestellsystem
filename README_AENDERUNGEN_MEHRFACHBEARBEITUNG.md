# Änderung: Mehrere Produkte auf einmal bearbeiten

Diese Version ergänzt im Adminbereich unter **Produkte** eine neue Mehrfachbearbeitung.

## Neu

- Produkte können per Checkbox ausgewählt werden.
- Alle sichtbaren Produkte können mit einem Klick ausgewählt werden.
- Für ausgewählte Produkte können gesammelt geändert werden:
  - Kategorie
  - Bezugsquelle
  - Status aktiv/deaktiviert
  - Sichtbarkeit für Standorte
- Ausgewählte Produkte können gesammelt gelöscht werden.

## Nutzung

1. Im Adminbereich **Produkte** öffnen.
2. In der Produktliste die gewünschten Produkte anhaken.
3. Im Bereich **Mehrere Produkte bearbeiten** die gewünschte Aktion auswählen.
4. Änderungen setzen.
5. Auf **Auf Auswahl anwenden** klicken.

Hinweis: Produktname, Gebindegröße und Bilder bleiben Einzelbearbeitung, damit nicht versehentlich viele individuelle Produktdaten überschrieben werden.
